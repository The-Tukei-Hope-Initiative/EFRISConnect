EFRISConnect

URA Electronic Fiscal Receipting & Invoicing for Manager.io

Fiscalise your Manager.io invoices and receipts with URA EFRIS

A free, open-source extension that connects Manager.io accounting software directly to Uganda's Electronic Fiscal Receipting and Invoicing Solution (EFRIS) - no separate system, no manual re-entry.

Open the Extension View on GitHub

Issue & Submit Receipts

Walk-in sales or existing Manager invoices, fiscalised to URA in one click - FDN, QR code and verification code included.

Credit Notes & Goods

File credit notes against original invoices, register goods/services with the correct EFRIS commodity codes, and manage stock.

Offline Mode

Documents queue automatically when URA is unreachable and submit as soon as the connection returns - or use URA's own Offline Enabler for true offline fiscalisation.

Multi-Business

Switch between multiple businesses in the same Manager.io install, each with its own EFRIS credentials and settings.

Add this to Manager.io

  1. In Manager.io, go to Settings → Custom Buttons/Extensions.
  2. Paste this URL as the extension address:
/extension
Once added, the extension opens inside Manager for whichever business you're viewing.
